# Бушуев Никита БАС-1 Лабораторная №7

# Задание 1 (Задачи 1–5)

## Задача 1
### Текст задачи
В текстовом файле хранятся целые числа по одному в строке. Найти сумму квадратов элементов. Исходный файл заполнить случайными данными отдельным методом.
### Алгоритм решения
1. Метод генерации создает текстовый файл и записывает в него случайные целые числа с использованием StreamWriter.
2. Основной метод открывает файл через StreamReader и считывает его построчно.
3. Каждая строка преобразуется в число, возводится в квадрат и суммируется.

## Задача 2
### Текст задачи
В текстовом файле хранятся целые числа по несколько в строке. Вычислить произведение элементов. Исходный файл заполнить случайными данными отдельным методом.
### Алгоритм решения
1. Генерируется файл, где числа в строках разделены пробелами.
2. При чтении каждая строка разбивается на массив строк методом Split.
3. Полученные элементы парсятся в int, после чего вычисляется их общее произведение.

## Задача 3
### Текст задачи
В текстовом файле хранится текст. Переписать в другой файл строки, имеющие заданную длину m.
### Алгоритм решения
1. Файл считывается построчно.
2. Для каждой строки проверяется ее длина (свойство Length).
3. Если длина строки равна m, она записывается в новый файл через StreamWriter.

## Задача 4
### Текст задачи
Бинарный файл содержит числовые данные. Вычислить произведение нечетных отрицательных компонент файла. Заполнение организовать отдельным методом.
### Алгоритм решения
1. Создается бинарный файл, заполняемый числами через BinaryWriter.
2. Чтение осуществляется через BinaryReader.
3. Проверяется условие: число < 0 и число % 2 != 0. Все подходящие числа перемножаются.

## Задача 5
### Текст задачи
Информация о багаже (название и масса) хранится в бинарном файле (структуры). Использовать XML-сериализацию. Найти число пассажиров, имеющих более двух единиц багажа, и число пассажиров, количество единиц багажа которых превосходит среднее.
### Алгоритм решения
1. Созданы структуры BaggageItem и Passenger.
2. Используется XmlSerializer для сохранения массива пассажиров в файл и его последующей загрузки.
3. Рассчитывается среднее количество вещей на человека.
4. Проходом по коллекции вычисляется количество пассажиров, соответствующих обоим условиям задачи.

# Задание 2 (Задачи 6–10)

## Задача 6
### Текст задачи
Составить программу, которая в списке L заменяет первое вхождение списка L1 на список L2. Использовать List.
### Алгоритм решения
1. В основном списке L ищется последовательность элементов, полностью совпадающая с L1.
2. При нахождении первого вхождения элементы удаляются методом RemoveRange.
3. На место удаленных элементов вставляется список L2 методом InsertRange.

## Задача 7
### Текст задачи
Сортировать элементы списка по возрастанию. Использовать LinkedList.
### Алгоритм решения
1. Данные хранятся в двусвязном списке LinkedList.
2. Реализована сортировка пузырьком: программа итерируется по узлам LinkedListNode, сравнивая значения текущего и следующего узлов.
3. Если текущее значение больше следующего, они меняются местами.

## Задача 8
### Текст задачи
На основе перечня игр и данных о студентах определить: в какие игры играют все; в какие — некоторые; в какие — никто. Использовать HashSet.
### Алгоритм решения
1. Все списки игр преобразуются в HashSet для быстрой работы с множествами.
2. Метод IntersectWith используется для нахождения пересечения (игры, в которые играют все).
3. Метод ExceptWith используется для нахождения разности (исключение тех игр, в которые играет хотя бы один студент, из общего перечня).

## Задача 9
### Текст задачи
Текст на русском языке. Напечатать в алфавитном порядке все глухие согласные буквы, которые не входят хотя бы в одно слово. Использовать HashSet.
### Алгоритм решения
1. Определен эталонный массив глухих согласных.
2. Текст разбивается на слова. Для каждой буквы из эталона проверяется, содержится ли она в каждом конкретном слове.
3. Если буква отсутствует хотя бы в одном слове, она сохраняется в HashSet.
4. Результат выводится на экран в алфавитном порядке.

## Задача 10
### Текст задачи
Сведения о результатах олимпиады. Напишите программу, которая будет выводить на экран фамилию и имя участника, набравшего максимальное количество баллов. Использовать SortedList или Dictionary.
### Алгоритм решения
1. Данные читаются из файла и суммируются для каждого участника.
2. Используется SortedList, где ключом является сумма баллов, а значением — список фамилий.
3. SortedList автоматически сортирует баллы по возрастанию.
4. Программа обращается к последнему элементу списка (самый высокий балл) и выводит всех участников с этим результатом.
